・ゲーム名	決めろ対掌衝撃波
・仕様ツール	Visual Studio 2017
・制作期間	6ヶ月
・制作人数	1人
・制作環境	WindowsPC

・自分が制作にかかわった部分
このゲームの全ての部分。

・このゲームについて
　このゲームは私が初めて制作したゲームです。制作した経緯は2年生の時に入学した1年生を歓迎するイベントがあったのですがそこに展示するために同学年のプランナーから制作を依頼されました。
　この制作では「DXライブラリ」を使用しました。

・遊び方
このゲームは2人で行うゲームでプレイヤーは制限時間内に発射台より玉を発射し、玉をたくさん中央の穴に入れてスコアを競うゲームです。発射台はゲーム画面の右下、左下にあり、角度を画面の内側に最大90度向けることが出来ます。発射台から玉を発射するときにスタミナを消費します。スタミナを全て消費してしまうとしばらく玉を発射できなくなります。玉は発射されると少し直進した後に螺旋状に回転し、徐々に中央に移動します。お互いの玉が重なると通常の玉より高得点の玉に合体します。玉の軌道は制限時間が半分になると逆方向に変わります。制限時間が無くなるとスコアに応じた結果が描画されます。

・操作説明
 タイトル
　Spaceキー,ゲーム開始
　→キー,開発スタッフ選択
　←キー,操作指南選択
　Enterキー,→キー、←キーで選択した画面の表示

 ゲーム
　→キー,発射台を時計回りに回転
　←キー,発射台を反時計回りに回転
　Enterキー,玉を発射
　Escキー,ゲーム終了

 リザルト
　→キー,タイトルを選択
　←キー,再挑戦を選択
　Enterキー,決定

 操作指南、開発スタッフ画面
　BackSpaceキー,タイトルに戻る
