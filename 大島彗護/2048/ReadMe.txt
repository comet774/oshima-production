・ゲーム名	2048
・使用ツール	Visual Studio 2017
・制作期間	1ヶ月
・制作人数	1人
・制作環境	WindowsPC

・自分が制作にかかわった部分
このゲームの全ての部分。

・このゲームについて
　このゲームの元は外国人の方が制作されたゲームで、そのゲームを真似て制作した。元となったゲームはPlayストア等で無料インストールができます。　

・制作動機
このゲームを制作しようとした当時、私はゲーム制作を行っておらず、何かplayストア等にあるゲームを短期間で制作出来ないかと考え、このゲームを選び、制作しました。

・遊び方
　ゲーム画面には4×4の盤面とスコアが表示されている。最初盤面のどこかに2,または4が書かれたブロックが二つ出現します。プレイヤーはブロックをキーで移動させ、同じ数字のブロック同士をぶつけると合体します。この時にスコアが発生します。同じ数字のブロックを合体させると「元のブロックの数字×2」の数字のブロックができます。ブロックを動かすと盤面の空いている場所に新しいブロックが1つ出現します。最終的に盤面が全て埋まりかつ動かせなくなるとゲームオーバーで、ゲームオーバーになるまでに稼いだスコアを競うゲームとなっています。

・操作説明
 ↑キー,全てのブロックを上に移動
 ↓キー,全てのブロックを下に移動
 →キー,全てのブロックを右に移動
 ←キー,全てのブロックを左に移動
 Escキー,ゲーム終了
