・仕様ツール	Visual Studio 2017
・制作期間	5ヶ月( 現在も制作中 )
・制作人数	1人
・制作環境	WindowsPC

・自分が制作にかかわった部分
このゲームの全ての部分。

・この制作物について
　この制作物はExcelで制作したcsvファイルの内容をゲームに反映する目的で制作したものです。csvファイルの内容は敵のステータス、形、色、出現順番となります。
　ゲームの内容はターン制のコマンドバトルにダンジョンのように階層があり、それをクリアしていくゲームとなっています。要はドラゴンクエストのバトルシステムにパズドラのようなダンジョン要素を取り入れ、連続戦っていき、全ての敵を倒すとクリアとなるゲームになっています。ドラゴンクエストの戦闘システムを指定された回数繰り返すゲームと考えてもらえれば良いです。
　描画、キーボード入力にDXライブラリを使用してます。

・遊び方
　ゲームはコマンドを選んで戦う戦闘を繰り返し、全ての敵を倒すとゲームクリアとなる。プレイヤーは「攻撃」、「防御」のどちらかを選択する。攻撃を選ぶと敵を選択し敵にダメージを与える。防御は敵に攻撃しない代わりに自身の防御力を上げる。選択後、お互いの行動を行う。行動は必ずプレイヤーから行う。その場の敵を全て倒すと次の敵が出てくる。プレイヤーの体力が0になるとゲームオーバーとなる。

・操作説明

↑キー,選択カーソルを上に移動
↓キー,選択カーソルを下に移動
Enterキー,決定
BackSpaceキー,一つ前の選択画面に戻る
Escキー,ゲーム終了
